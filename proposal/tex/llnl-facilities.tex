
Lawrence Livermore National Laboratory provides world-leading computing resources through the Liv-
ermore Computing Center, including leadership-class supercomputers, advanced data infrastructure, and
specialized platforms that support large-scale simulation, data science, and machine learning workflows.
Allocations are available through the Advanced Simulation and Computing (ASC) program, the Institutional
Computing (IC) program, and Director’s Discretionary allocations.
%
{\bf El Capitan} is LLNL’s first exascale supercomputer, based on the HPE Cray EX platform and powered by
AMD Instinct MI300A accelerated processing units (APUs), designed to support large-scale multiphysics
simulation, AI-assisted modeling, and high-fidelity predictive science at exascale.
%
{\bf Sierra} is a leadership-class hybrid CPU/GPU system comprising IBM POWER9 processors and NVIDIA
V100 GPUs that supports large-scale HPC simulation, machine learning, and data-intensive workflows,
serving as a critical production platform for multiphysics and AI-enabled scientific applications.
%
{\bf Tuolumne} is a next-generation advanced technology system designed to provide pre-exascale capability
and early access to emerging architectures, supporting code development, performance portability testing,
and scaling studies for next-generation HPC systems.
